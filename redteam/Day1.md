* OSI Layer
| Layer        | Fundamental Question It Answers |
| ------------ | ------------------------------- |
| Physical     | Can electrons/light move?       |
| Data Link    | Can I talk to the next device?  |
| Network      | Can I reach a distant network?  |
| Transport    | Can I talk reliably or fast?    |
| Session      | Is this conversation ongoing?   |
| Presentation | How is data represented?        |
| Application  | What does the user want?        |

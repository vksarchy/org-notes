Go with **elements** for functional components and **assets** for high-value resources.

### Example:

- **Elements:** Tactical sub-units performing specific tasks.
    
    - Legs → "Mobility Elements"
    - Arms → "Manipulation Elements"
    - Lungs → "Respiration Element"
    - Kidneys → "Filtration Element" (Now you can tell them to push harder.)
- **Assets:** Critical resources you don’t want to lose.
    
    - Eyes → "Optical Assets"
    - Hands → "Dexterity Assets"
    - Brain → "Command & Control Asset"
    - Stomach → "Processing Asset"

### Command Example:

_"Filtration Element, increase output—mission demands peak efficiency."_

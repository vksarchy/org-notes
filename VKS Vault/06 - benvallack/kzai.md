# 30 Key Keyboard Layout

#idea

A layout based on using the inner column bottom row keys as thumb
keys. This makes for a super simple layout of 5x3 with no extra
thumbs.


bldwv kfouj
nrtsg yhaei
qxmc* *p.,/

(* * are the two thumb keys, space and shift)

# nvim config tip for grep search

One of the best things about using @joseanmartinez excellent nvim walkthtrough
is the [Telescope](https://github.com/nvim-telescope/telescope.nvim)
plugin with shortcuts to search with grep. `Leader f s` will search
all files in current working directory. Useful for coding and
[[t63i|notetaking]].

[Josean Martinez excellent full walkthrough](https://www.josean.com/posts/how-to-setup-neovim-2024)

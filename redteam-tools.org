* Introduction

This Org-mode document contains a comprehensive list of red team and penetration testing tools for Arch Linux. Each section contains executable bash code blocks that can be run individually or combined into a complete installation script.

** Usage Instructions

1. To execute a single code block: =C-c C-c= with cursor in the block
2. To execute all blocks: =C-c C-v b= (org-babel-execute-buffer)
3. To tangle all blocks to a script: =C-c C-v t= (org-babel-tangle)

* System Preparation

** Update System
#+begin_src bash :tangle redteam-install.sh
#!/bin/bash
# Red Team Tools Installation Script for Arch Linux
# Generated from redteam-tools.org

set -e  # Exit on error

echo "[*] Updating system..."
sudo pacman -Syu --noconfirm
#+end_src

** Install Base Development Tools
#+begin_src bash :tangle redteam-install.sh

sudo pacman -S --noconfirm base-devel git curl wget python python-pip python-pipx \
    go rust ruby ruby-bundler nodejs npm cmake make gcc clang
#+end_src

** Install AUR Helper (yay)
#+begin_src bash :tangle redteam-install.sh
echo "[*] Installing yay AUR helper..."
if ! command -v yay &> /dev/null; then
    cd /tmp
    git clone https://aur.archlinux.org/yay.git
    cd yay
    makepkg -si --noconfirm
    cd ..
    rm -rf yay
fi
#+end_src
